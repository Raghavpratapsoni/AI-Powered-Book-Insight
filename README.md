# 📚 AI Book Intelligence Platform

An end-to-end full-stack web application that uses AI and Retrieval-Augmented Generation (RAG) to process books, generate insights, and answer user queries intelligently.

This platform automates book data collection, stores it efficiently, and allows users to interact with the data using natural language.
